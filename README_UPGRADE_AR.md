# TrustPay Pro Upgrade

هذه نسخة مطورة من المشروع اليدوي الحالي وتضيف:

- Mini App على `/app?tid=TELEGRAM_ID`
- لوحة إدارة على `/admin/dashboard`
- إدارة طرق الدفع
- أسعار USDT وحاسبة
- طلبات إيداع وسحب مع إثباتات
- KYC
- تذاكر دعم
- إحالات ونقاط
- إشعارات
- كشف PDF بسيط

## طريقة التحديث اليدوي

1. فك ضغط الملف.
2. ارفع الملفات إلى نفس مستودع GitHub الحالي `trustpay-cloud`.
3. وافق على استبدال الملفات القديمة.
4. اعمل Commit changes.
5. في Render اضغط:
   Manual Deploy → Deploy latest commit

## روابط بعد النشر

- لوحة الإدارة:
  `/admin/dashboard`
- Mini App:
  `/app?tid=رقم_تلجرام`
- Health:
  `/health`

## ملاحظات مهمة

- الحسابات البنكية موجودة كبيانات أولية في قاعدة البيانات عند أول تشغيل.
- يمكن إضافة/تعطيل طرق الدفع من لوحة الإدارة.
- لا تضع التوكنات داخل الملفات؛ اتركها في Render Environment Variables.
- إذا كنت تقدم خدمات تحويل أو بيع/شراء عملات رقمية، تأكد من الالتزام بالأنظمة والتراخيص في بلدك.
